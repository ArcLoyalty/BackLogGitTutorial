파일 커밋(Commit)하기

	utorial 폴더에 새로운 파일을 추가하고, 원격 저장소에 파일을 등록해본다.

	우선 tutorial 폴더 안에 「sample.txt」라는 이름으로 텍스트 파일을 만든다. 
	파일 내용에는 적당히 아래 텍스트를 입력하면 된다.

		원숭이도 이해할 수 있는 Git 명령어(이해하지 못해서 미안하다 쒸불럼아)

	tutorial 폴더를 열어, 폴더 안을 우클릭해 우클릭 메뉴를 표시해준다. 그 우클릭 메뉴에서 Git 커밋을 클릭한다.

	아래 화면이 표시되면, 변경한 항목에서 sample.txt를 선택해 메시지란에 커밋 메시지를 입력한 후 OK 버튼을 클릭한다.

	아래 화면처럼 성공이라고 표시될 거다. 클로즈 버튼을 클릭해 화면을 닫아준다.

	우클릭 메뉴의 TortoiseGit> 로그 표시를 클릭한다. 방금 전의 커밋이 이력에 추가되어 있을 거다	

	(이거 업로드 아니야 그냥 업데이트 목록에 추가하는 거야.)
