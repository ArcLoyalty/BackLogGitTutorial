Backlog에 원격 저장소 생성하기

	우선은 원격 저장소를 Backlog 상에 만들어보자.(Git에 만드는거 아니였어??)

	Backlog 에서는 관리자 권한이 있는 사용자만 Git 저장소를 만들 수 있다. 
	만약 관리자 권한을 가지고 있지 않으면 관리자 권한을 가지고 있는 사람에게 저장소 생성을 요청하거나 무료 계정을 만들어(30일 동안 무료 사용 가능) Backlog 공간을 이용할 수도 있다.

	Git 을 만드려면 최소 1개의 프로젝트가 필요하다.

	상단 메뉴에서 Git 메뉴가 보이지 않을 경우, 프로젝트를 선택 후 「프로젝트 설정」 > 「Git 설정」 에서 Git 기능을 활성화시킨다.

	Git 기능을 활성화 한 후, 동일한 Git의 설정 화면에서 「저장소를 추가한다」를 클릭한다.

	「저장소 이름」과 「설명」을 입력해 「저장소를 작성한다」 버튼을 클릭한다. 여기서는 이름에 「tutorial」, 설명에 「튜토리얼용입니다」라고 입력한다.

	아래 그림처럼 새로운 저장소가 추가되었는지 확인한다.

	(자 안보일 수도 있어. 근데 프로젝트부터 만들고난 후에 프로젝트 설정창을 보면 Git이 있는데 disenable되있을거야 ㅇㅁ)
